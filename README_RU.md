# mod-gearscore

Модуль для AzerothCore, который рассчитывает и сохраняет GearScore (уровень экипировки) игроков в базу данных персонажей. Для обеспечения 100% точности с игрой, модуль использует те же математические формулы, что и популярный аддон TacoTip.

## Особенности
- Автоматически рассчитывает GearScore при входе в игру, а также при надевании и снятии предметов.
- Сохраняет значение в отдельную таблицу `character_gearscore` базы данных `acore_characters` (включая класс и имя персонажа для удобства чтения).
- Содержит внутриигровую команду `.gs` для проверки своего GS или GS другого игрока (как онлайн, так и оффлайн).
- Встроенная защита от ботов: не тратит процессорное время и место в базе на ботов (настраивается в конфиге).
- Настройка экспорта в CSV (подготовка для внешних API Google Таблиц).
- Все настройки вынесены в `gearscore.conf`.

## Внутриигровые команды
- `.gs` - Показывает ваш собственный GearScore или GS выбранной вами цели (если вы выделили игрока). Если в таргете NPC — показывает ваш собственный GearScore.
- `.gs [ИмяИгрока]` - Показывает GearScore конкретного игрока по имени (берет из базы данных, даже если он оффлайн).
- `.gs help` - Показывает справку по использованию команды (на русском языке, если локаль клиента ruRU).

## Установка
1. Поместите папку `mod-gearscore` в директорию `modules/` вашего сервера.
2. Пересоберите сервер (используйте `cmake` и ваш компилятор).
3. Импортируйте SQL-файл из `sql/characters/base/character_gearscore.sql` в вашу базу данных `acore_characters`.
4. Скопируйте файл `conf/gearscore.conf.dist` в `conf/gearscore.conf` и настройте по желанию.

## API / Экспорт (Google Таблицы)
Благодаря тому, что все данные GearScore лежат в удобной и компактной таблице `character_gearscore`, вы можете легко написать внешний скрипт (например, на PHP или Python) для выдачи этих данных в формате CSV по токену, чтобы автоматически подтягивать их в Google Таблицы.
